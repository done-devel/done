## Dependencies to build
- gtk4
- libadwaita
- pkg-config

Ubuntu 22.04:
```bash
sudo apt install libadwaita-1-dev libgtk-4-dev libsqlite3-dev
```
Arch Linux:
```bash
sudo pacman -S libadwaita gtk4 sqlite
```
